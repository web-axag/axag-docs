# Manifest Field Reference

Complete reference for all manifest fields. See [Manifest Schema](/docs/semantic-manifest/manifest-schema) for the structural overview.

For each field, this reference provides: name, type, required status, description, constraints, and examples.

Refer to the [Full Schema Reference](/docs/reference/schema-reference) for the canonical JSON Schema document.
