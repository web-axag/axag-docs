# Annotating Composite Flows

Composite flows combine multiple operations into a logical workflow. Each step is annotated individually with workflow attributes linking them.

Use `axag-workflow-id` to group steps and `axag-workflow-step` to declare order. Preconditions and postconditions chain steps automatically.
